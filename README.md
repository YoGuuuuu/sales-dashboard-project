# 📊 Sales Dashboard Project

## 🚀 Overview
This project is an interactive Sales Dashboard built using Power BI and SQL.  
It helps analyze revenue trends, category performance, and top customers/products.

---

## 🛠 Tools Used
- Power BI
- SQL (MySQL)
- Excel

---

## 📈 Key Insights
- Technology category generated the highest sales
- Sales peak observed in November and December
- Top 5 customers contributed major revenue share

---

## 📊 Features
- KPI Card (Total Revenue)
- Sales by Category (Bar Chart)
- Sales Trend by Month (Line Chart)
- Top 5 Customers Analysis
- Year Filter (Slicer)

---

## 📂 Files Included
- `SalesAnalysis.pbix` → Power BI Dashboard  
- `Sales.sql` → SQL Queries  
- `Sales.png` → Dashboard Screenshot  
- `sales_datasets.zip` → Dataset  

---

## 📌 How to Use
1. Download `.pbix` file  
2. Open in Power BI Desktop  
3. Explore dashboard and filters
